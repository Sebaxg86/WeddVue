# WeddVue 🚀

Private QR-powered wedding photo platform for multi-event galleries and guest uploads.

## 🌐 Live Demo

> [Click here to view the live site](https://weddvue.vercel.app)

## ⚙️ Technologies Used

- React 19
- TypeScript
- Vite
- Supabase

## 📚 Description

This project was developed as part of a real product build for a wedding pilot and includes:

- ✅ Multi-event private dashboard for couples
- ✅ QR-based guest photo uploads
- ✅ Private gallery with favorites, downloads, and ZIP export
- 📖 Documentation: [Docs/README.md](Docs/README.md)
