# WedSnap 🚀

A private QR-powered wedding photo collection platform for capturing the event from every guest's perspective.

## 🌐 Live Demo

> Live demo coming soon. Deployment will be added once the MVP is ready.

## ⚙️ Technologies Used

- React + Vite
- Supabase
- Vercel
- JavaScript

## 📚 Description

This project was developed as a personal product challenge to build a frictionless wedding photo upload experience and includes:

- ✅ QR-based guest uploads by table
- ✅ Private photo storage with admin-only access
- ✅ Batch uploads with device-level traceability
